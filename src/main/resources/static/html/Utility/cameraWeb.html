얼굴 인식 서비스

얼굴 인식 서비스를 활용하여 나와 비슷한 연예인과 나이를 분석
비슷한 사람들의 도서목록을 추천 받을 수 있습니다.

이용 방법

1. 카메라 켜기 [버튼]을 클릭합니다.
2. 얼굴이 잘보이게 위치해야 합니다.
3. 사진 전송하기 [버튼]을 클릭합니다.
4. 잠시후 추천 도서 목록을 제공 받습니다.
이 문장은 사용자의 웹 브라우저가 canvas 요소를 지원하지 않을 때 나타납니다!
